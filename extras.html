Emojis

Computers don’t know english... They only “speak” binary (0s and 1s). So in order to use the alphabet they have each character saved in a place in the computer’s memory - this is referred to as an address - as a special sequence of binary code (decimal / hexadecimal) . In order to print the emojis we need to point to the address of the emoji in memory.
